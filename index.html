Introduction

This project explores our dataset (cleaned.csv) with multiple charts. We can switch among quarters (Q1..Q5) for selected charts, and we have an aggregated line chart and stacked bar chart for Q1..Q5.

Data Information

cleaned.csv includes columns like: OPE ID, School, State, Zip Code, School Type, Dependent Students_Q1, Independent Students_Q1, Quarterly Total_Q1, ... (and similar columns for Q2..Q5).

  • Chart 1: Bar Chart – Top 10 schools by the selected quarter’s total.
  • Chart 2: Line Chart – Aggregated totals for Q1..Q5.
  • Chart 3: Scatter Plot – Dependent vs. Independent for selected quarter.
  • Chart 4: Pie/Donut – School Type by selected quarter.
  • Chart 5: Stacked Bar – Summed Dep vs. Ind for Q1..Q5.
  • Chart 6: Map Visualization – By selected quarter’s total.

Conclusion

This version does not incorporate forecast data, focusing solely on cleaned.csv. We can switch quarters for bar, scatter, pie, and map charts, plus view aggregated line and stacked bar charts.
